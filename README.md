# Pandilla
## Un proyecto para el refugio Pandilla Animal

### Acerca de 
Este proyecto es un gestor para administrar las adopciones, transitos y rifas que lleva adelante el refugio

### Tecnologías
- Springboot
- Angular
-  Mysql
-  Firebase
